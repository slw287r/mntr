# Monitor process and its supprocesses' MEM and CPU Usage

* MEM: RSS and SHR in g
* CPU: Total in %

## Requirements
- Linux OS
- procps

## Usage
- by process name

```
mntr <cmd_name>
```

- by process pid
```
mntr <process_pid>
```

- by run a program
```
mntr <cmd> <args> ...
```
